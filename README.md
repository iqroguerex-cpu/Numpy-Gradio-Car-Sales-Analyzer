## 🚀 Overview

A **Car Sales Analytics Dashboard** built with **Streamlit**, designed to analyze sales performance across multiple categories and salespeople.

This project simulates real-world business analytics scenarios such as performance tracking, ranking, and category insights.

---

## ✨ Features

* 🚗 Generate multi-category sales data
* 📊 Salesperson performance analysis
* 📈 Category-wise sales insights
* 🏆 Ranking system for top performers
* 🔍 Filter high-performing salespeople
* 📊 Interactive charts for visualization

---

## 🛠️ Tech Stack

* Python 3.x
* Streamlit
